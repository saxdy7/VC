## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)

## Usage

1. Enter your name on the home page
2. Click "New Meeting" to create a room or enter a Room ID to join an existing meeting
3. Share the URL with others to invite them
4. Control your camera, microphone, and other settings from the meeting interface

## Environment

No environment variables needed - uses ZegoCloud test credentials.
